### AgoraRTCProvider

| 属性名     | 类型              | 默认值 | 描述                                                                                                                                          |

| `children` | `ReactNode`       | -      | 需要展示的 React 节点。                                                                                                                       |

#### 使用示例

```jsx
import { AgoraRTCProvider } from "agora-rtc-react";
import AgoraRTC from "agora-rtc-sdk-ng";

function App({ children }) {
  const [client] = useState(() => AgoraRTC.createClient({ mode: "rtc", codec: "vp8" }));
  return <AgoraRTCProvider client={client}>{children}</AgoraRTCProvider>;
}
```

### LocalAudioTrack

#### Props

| 属性名     | 类型               | 默认值 | 描述                                                                                                                                                                                                     |


### RemoteVideoPlayer

该组件用于订阅和播放远端用户的视频轨道。

#### Props

| 属性名      | 类型                          | 默认值                                                                                                                                                     | 描述                                                                                                                                                                |
| ----------- | ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `track`     | `IRemoteVideoTrack`           | -                                                                                                                                                          | 远端视频轨道对象。详见 [IRemoteVideoTrack](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iremotevideotrack.html)。 |
| `playVideo` | `boolean`                     | [user.hasVideo](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartcremoteuser.html#hasvideo) 属性的值 | <li>`true`：播放远端用户的视频轨道。</li><li>`false`：停止播放远端用户的视频轨道。</li>                                                                             |
| `cover`     | `string` 或 `() => ReactNode` | -                                                                                                                                                          | 当 `playVideo` 为 `false`时要渲染的封面图片或自定义组件，用于替代视频画面显示。                                                                                     |
| `client`    | `IAgoraRTCClient`             | -                                                                                                                                                          | [IAgoraRTCClient](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartcclient.html) 对象。                       |
| `children`  | `ReactNode`                   | -                                                                                                                                                          | 需要展示的 React 节点。                                                                                                                                             |

#### 注意事项

一个 `IRemoteVideoTrack` 对象只能由一个 `RemoteVideoPlayer` 拥有。

#### 使用示例

```jsx
import { RemoteUser, useRemoteUsers, useRemoteVideoTracks } from "agora-rtc-react";

function App() {
  //get remote users
  const remoteUsers = useRemoteUsers();
  //get remote video tracks by remote users
  const videoTracks = useRemoteVideoTracks(remoteUsers);

  return (
    <>
      {videoTracks.map(track => (
        <RemoteVideoPlayer key={track.getUserId()} track={track} />
      ))}
    </>
  );
}
```
