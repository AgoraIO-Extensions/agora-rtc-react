### IAgoraRtcReactError

抛出的错误。

```typescript
export interface IAgoraRtcReactError extends AgoraRTCError {
  readonly rtcMethod: string;
  readonly rtcError: AgoraRTCError | string;
}
```

rtcmethod: 报错的 web 方法是什么；rtcerror 是完全对齐 web 的 AgoraRTCError

| 属性        | 类型                                    | 是否必选 | 描述                       |
| ----------- | --------------------------------------- | -------- | -------------------------- |
| `rtcMethod` | `string`                                | 必选     | 报错的 JavaScript 方法。   |
| `rtcError`  | `AgoraRTCError` &VerticalLine; `string` | 必选     | web 的 class AgoraRTCError |
