## useRTCClient

用于获取 `IAgoraRTCClient` 对象。

#### 参数

| 参数名   | 类型                                                                                          | 是否必选 | 描述                                                                                                                                                                                                                 |
| -------- | --------------------------------------------------------------------------------------------- | -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `client` | [`IAgoraRTCClient`](/api-ref/rtc/react/interfaces/iagorartcclient.html) &VerticalLine; `null` | 可选     | 如果传入该参数，则使用传入的 `IAgoraRTCClient` 对象；如果不传入该参数，则使用从[父组件的 Context](https://doc.shengwang.cn/api-ref/rtc/react/react-sdk/components#agorartcprovider)中获取的 `IAgoraRTCClient` 对象。 |

#### 返回值

| 类型                                                                    | 描述                     |
| ----------------------------------------------------------------------- | ------------------------ |
| [`IAgoraRTCClient`](/api-ref/rtc/react/interfaces/iagorartcclient.html) | `IAgoraRTCClient` 对象。 |

#### 使用示例

```jsx
import { useRTCClient } from "agora-rtc-react";

function App() {
  const client = useRTCClient();

  return <></>;
}
```
