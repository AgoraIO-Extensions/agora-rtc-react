### useAutoPlayVideoTrack

用于在组件卸载时停止播放本地或远端视频轨道的 Hook。// TODO: 那当组件挂载时时默认开始播放吗？

如果祖先元素中存在 `<TrackBoundary />`，则 `useAutoPlayVideoTrack` 不会在组件卸载时停止播放轨道，而是委托给 TrackBoundary。// TODO: TrackBoundary 组件之前说不用对外，那这里怎么描述比较好；另外“祖先元素”是不是叫“父节点”更好？

#### 参数

| 参数名  | 类型               | 是否必选          | 描述                                 |
| ------- | ------------------ | ----------------- | ------------------------------------ | ------------------------------------------------------------------------------------------------------ |
| `track` | `IRemoteVideoTrack | ILocalVideoTrack` | 必选                                 | 远端视频轨道对象或本地视频轨道对象。                                                                   |
| `play`  | `boolean`          | 可选              | 是否播放视频轨道。//TODO: 有默认值吗 |
| `div`   | `HTMLElement       | null`             | 可选                                 | 用于渲染视频的 HTML 元素。如果提供了该参数，视频将在该元素中自动播放。如果未提供，则不会自动播放视频。 |

#### 返回值

无。
