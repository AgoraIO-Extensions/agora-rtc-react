### useVolumeLevel

用于获取音频轨道音量级别的 Hook。

#### 参数

| 参数名       | 类型               | 是否必选         | 描述       |
| ------------ | ------------------ | ---------------- | ---------- | ---- | ------------------------------------------------ |
| `audioTrack` | `IRemoteAudioTrack | ILocalAudioTrack | undefined` | 可选 | 本地或远端音频轨道。如果未定义，则音量级别为 0。 |

#### 返回值

| 类型     | 描述                                |
| -------- | ----------------------------------- |
| `number` | 音频轨道的音量级别，范围从 0 到 1。 |
