### useIsConnected

用于获取客户端是否连接到服务器的 Hook。

#### 参数

| 参数名   | 类型                                         | 是否必选 | 描述                     |
| -------- | -------------------------------------------- | -------- | ------------------------ |
| `client` | `IAgoraRTCClient `<code>&#124;</code>` null` | 可选     | `IAgoraRTCClient` 对象。 |

#### 返回值

| 类型      | 描述                                                                              |
| --------- | --------------------------------------------------------------------------------- |
| `boolean` | <li>`true`：客户端已连接到服务器。</li><li>`false`：客户端没有连接到服务器。</li> |

#### 使用示例

```jsx
import { useIsConnected } from "agora-rtc-react";

function App() {
  const isConnected = useIsConnected();

  return <div>{isConnected}</div>;
}
```
