## AgoraRTCProvider

该组件用于将传入的 `client` 对象提供给 `children` 内的各个组件，即向子组件提供 [Context](https://react.dev/learn/passing-data-deeply-with-context)。

#### Props

| 属性名     | 类型              | 默认值 | 描述                                                                                                                                          |
| ---------- | ----------------- | ------ | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `client`   | `IAgoraRTCClient` | 无     | [IAgoraRTCClient](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartcclient.html) 对象。 |
| `children` | `ReactNode`       | 无     | 需要展示的 React 节点。                                                                                                                       |

#### 使用示例

```jsx
import { AgoraRTCProvider } from "agora-rtc-react";
import AgoraRTC from "agora-rtc-sdk-ng";

function App({ children }) {
  const [client] = useState(() => AgoraRTC.createClient({ mode: "rtc", codec: "vp8" }));
  return <AgoraRTCProvider client={client}>{children}</AgoraRTCProvider>;
}
```

## AgoraRTCScreenShareProvider

该组件用于将传入的屏幕共享 `client` 对象提供给 `children` 内的各个组件。

#### Props

| 属性名     | 类型              | 默认值 | 描述                                                                                                                                          |
| ---------- | ----------------- | ------ | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `client`   | `IAgoraRTCClient` | 无     | [IAgoraRTCClient](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartcclient.html) 对象。 |
| `children` | `ReactNode`       | 无     | 需要展示的 React 节点。                                                                                                                       |

#### 注意事项

你可以同时使用 `AgoraRTCScreenShareProvider` 和 `AgoraRTCProvider` ，但二者不会共享 `client` 属性。

#### 使用示例

```jsx
import { AgoraRTCScreenShareProvider } from "agora-rtc-react";
import AgoraRTC from "agora-rtc-sdk-ng";

function App({ children }) {
  const [client] = useState(() => AgoraRTC.createClient({ mode: "rtc", codec: "vp8" }));
  return <AgoraRTCScreenShareProvider client={client}>{children}</AgoraRTCScreenShareProvider>;
}
```

## LocalAudioTrack

该组件用于播放本地音频轨道（不支持指定使用的媒体设备）。

#### Props

| 属性名     | 类型               | 默认值  | 描述                                                                                                                                                                                                     |
| ---------- | ------------------ | ------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `track`    | `ILocalAudioTrack` | 无      | 要播放的本地音频轨道。通过 [createMicrophoneAudioTrack](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartc.html#createmicrophoneaudiotrack) 创建。 |
| `play`     | `boolean`          | `false` | <li>`true`：播放该轨道。</li><li>`false`：停止播放该轨道。</li>                                                                                                                                          |
| `volume`   | `number`           | 无      | 音量大小。取值范围 [0, 1000]，0 代表静音，100 代表原始音量。100 以上会使用 [Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API) 进行音量增益。                                |
| `disabled` | `boolean`          | `false` | <li>`true`：禁用该轨道。禁用后，SDK 将停止播放和发布该轨道。</li><li>`false`：启用该轨道。</li>                                                                                                          |
| `muted`    | `boolean`          | `false` | <li>`true`：暂停发送该轨道的媒体数据。</li><li>`false`：恢复发送该轨道的媒体数据。</li>                                                                                                                  |
| `children` | `ReactNode`        | 无      | 需要展示的 React 节点。                                                                                                                                                                                  |

#### 注意事项

设置 `disabled` 和 `muted` 属性时其实是调用了 JavaScript SDK 的 [setEnabled](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/ilocaltrack.html#setenabled) 方法和 [setMuted](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/ilocaltrack.html#setmuted) 方法，因此：

- 与设置 `disabled` 属性相比，设置 `muted` 属性的响应速度更快且不影响音频采集状态。
- 不能同时设置 `muted` 属性和 `disabled` 属性。

#### 使用示例

```jsx
import { LocalAudioTrack, useLocalAudioTrack } from "agora-rtc-react";

function App() {
  const audioTrack = useLocalAudioTrack();
  return <LocalAudioTrack track={audioTrack} play />;
}
```

## LocalUser

该组件用于播放本地用户的摄像头视频轨道和麦克风音频轨道（不支持指定使用的媒体设备）。

当视频轨道停止播放时，该组件会展示封面图片。

#### Props

| 属性名       | 类型                    | 默认值  | 描述                                                                                                                                                                                                       |
| ------------ | ----------------------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `micOn`      | `boolean`               | `false` | <li>`true`：打开本地用户的麦克风。</li><li>`false`：关闭本地用户的麦克风。</li>                                                                                                                            |
| `cameraOn`   | `boolean`               | `false` | <li>`true`：打开本地用户的摄像头。</li><li>`false`：关闭本地用户的摄像头。</li>                                                                                                                            |
| `audioTrack` | `IMicrophoneAudioTrack` | 无      | 要播放的麦克风音频轨道。通过 [createMicrophoneAudioTrack](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartc.html#createmicrophoneaudiotrack) 创建。 |
| `videoTrack` | `ICameraVideoTrack`     | 无      | 要播放的摄像头视频轨道。通过 [createCameraVideoTrack](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartc.html#createcameravideotrack) 创建。         |
| `playAudio`  | `boolean`               | `false` | <li>`true`：播放本地用户的音频轨道。</li><li>`false`：停止播放本地用户的音频轨道。</li>                                                                                                                    |
| `playVideo`  | `boolean`               | `false` | <li>`true`：播放本地用户的视频轨道。</li><li>`false`：停止播放本地用户的视频轨道。</li>                                                                                                                    |
| `volume`     | `number`                | 无      | 音量大小。取值范围 [0, 1000]，0 代表静音，100 代表原始音量。100 以上会使用 [Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API) 进行音量增益。                                  |
| `cover`      | `string`                | 无      | 当 `playVideo` 为 `false`时要渲染的封面图片，用于替代视频画面显示。                                                                                                                                        |
| `children`   | `ReactNode`             | 无      | 需要展示的 React 节点。                                                                                                                                                                                    |

#### 使用示例

```jsx
import { LocalUser, useLocalAudioTrack, useLocalCameraTrack } from "agora-rtc-react";

function App() {
  const audioTrack = useLocalAudioTrack();
  const videoTrack = useLocalCameraTrack();

  return (
    <LocalUser
      audioTrack={audioTrack}
      cameraOn
      cover={COVER_IMAGE_URL}
      micOn
      playAudio
      playVideo
      videoTrack={videoTrack}
    />
  );
}
```

## RemoteAudioTrack

该组件用于播放远端用户的音频轨道，并且支持指定播放设备。

#### Props

| 属性名             | 类型                | 默认值 | 描述                                                                                                                                                                                                                                                                                                       |
| ------------------ | ------------------- | ------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `track`            | `IRemoteAudioTrack` | 无     | 要播放的远端音频轨道。详见 [IRemoteAudioTrack](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iremoteaudiotrack.html)。                                                                                                                                    |
| `play`             | `boolean`           | 无     | <li>`true`：播放该轨道。</li><li>`false`：停止播放该轨道。</li>                                                                                                                                                                                                                                            |
| `playbackDeviceId` | `string`            | 无     | 要使用的播放设备 ID，例如扬声器。设备 ID 通过 [getPlaybackDevices](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartc.html#getplaybackdevices) 获取。该属性仅支持桌面端的 Chrome 浏览器，在其它浏览器上修改该属性的值将会抛出 `NOT_SUPPORTED` 错误。 |
| `volume`           | `number`            | 无     | 音量大小。取值范围 [0, 100]，0 代表静音，100 代表原始音量。                                                                                                                                                                                                                                                |
| `children`         | `ReactNode`         | 无     | 需要展示的 React 节点。                                                                                                                                                                                                                                                                                    |

#### 使用示例

```jsx
import { RemoteAudioTrack, useJoin, useRemoteAudioTracks, useRemoteUsers } from "agora-rtc-react";

function App() {
  const remoteUsers = useRemoteUsers();
  const audioTracks = useRemoteAudioTracks(remoteUsers);

  return (
    <>
      {audioTracks.map(track => (
        <RemoteAudioTrack key={track.getUserId()} play track={track} />
      ))}
    </>
  );
}
```

## RemoteUser

该组件用于播放远端用户的视频和音频轨道，并且仅支持指定使用的音频设备、不支持指定使用的视频设备。

#### Props

| 属性名             | 类型                                      | 默认值                                                                                                                                                     | 描述                                                                                                                                                                                                                                                                                                       |
| ------------------ | ----------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `user`             | `IAgoraRTCRemoteUser`                     | 无                                                                                                                                                         | 远端用户对象。详见 [IAgoraRTCRemoteUser](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartcremoteuser.html)。                                                                                                                                        |
| `playVideo`        | `boolean`                                 | [user.hasVideo](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartcremoteuser.html#hasvideo) 属性的值 | <li>`true`：播放远端用户的视频轨道。</li><li>`false`：停止播放远端用户的视频轨道。</li>                                                                                                                                                                                                                    |
| `playAudio`        | `boolean`                                 | [user.hasAudio](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartcremoteuser.html#hasaudio) 属性的值 | <li>`true`：播放远端用户的音频轨道。</li><li>`false`：停止播放远端用户的音频轨道。</li>                                                                                                                                                                                                                    |
| `playbackDeviceId` | `string`                                  | 无                                                                                                                                                         | 要使用的播放设备 ID，例如扬声器。设备 ID 通过 [getPlaybackDevices](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartc.html#getplaybackdevices) 获取。该属性仅支持桌面端的 Chrome 浏览器，在其它浏览器上修改该属性的值将会抛出 `NOT_SUPPORTED` 错误。 |
| `volume`           | `number`                                  | 无                                                                                                                                                         | 音量大小。取值范围 [0, 100]，0 代表静音，100 代表原始音量。                                                                                                                                                                                                                                                |
| `cover`            | `string` &VerticalLine; `() => ReactNode` | 无                                                                                                                                                         | 当 `playVideo` 为 `false`时要渲染的封面图片或自定义组件，用于替代视频画面显示。                                                                                                                                                                                                                            |
| `children`         | `ReactNode`                               | 无                                                                                                                                                         | 需要展示的 React 节点。                                                                                                                                                                                                                                                                                    |

#### 使用示例

```jsx
import { RemoteUser, useRemoteUsers } from "agora-rtc-react";

function App() {
  const remoteUsers = useRemoteUsers();

  return (
    <>
      {remoteUsers.map(user => (
        <RemoteUser key={user.uid} user={user} />
      ))}
    </>
  );
}
```
