## NetworkQuality

上下行 last mile 网络质量。

```typescript
export interface NetworkQuality {
  uplink: 0 | 1 | 2 | 3 | 4 | 5 | 6;
  downlink: 0 | 1 | 2 | 3 | 4 | 5 | 6;
  delay: number;
}
```

| 属性       | 类型                                                                                                      | 是否必选 | 描述                                                                                                                                                                                                                                                                                                                                                    |
| ---------- | --------------------------------------------------------------------------------------------------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `uplink`   | `0 &VerticalLine; 1 &VerticalLine; 2 &VerticalLine; 3 &VerticalLine; 4 &VerticalLine; 5 &VerticalLine; 6` | 必选     | 上行网络质量。基于上行发送码率、上行丢包率、平均往返时延和网络抖动计算。<ul><li>0: 质量未知。</li><li>1: 质量极好。</li><li>2: 用户主观感觉和极好差不多，但码率可能略低于极好。</li><li>3: 用户主观感受有瑕疵但不影响沟通。</li><li>4: 勉强能沟通但不顺畅。</li><li>5: 网络质量非常差，基本不能沟通。</li><li>6: 网络连接断开，完全无法沟通。</li></ul> |
| `downlink` | `0 &VerticalLine; 1 &VerticalLine; 2 &VerticalLine; 3 &VerticalLine; 4 &VerticalLine; 5 &VerticalLine; 6` | 必选     | 下行网络质量。基于下行发送码率、下行丢包率、平均往返时延和网络抖动计算。<ul><li>0: 质量未知。</li><li>1: 质量极好。</li><li>2: 用户主观感觉和极好差不多，但码率可能略低于极好。</li><li>3: 用户主观感受有瑕疵但不影响沟通。</li><li>4: 勉强能沟通但不顺畅。</li><li>5: 网络质量非常差，基本不能沟通。</li><li>6: 网络连接断开，完全无法沟通。</li></ul> |
| `delay`    | `number`                                                                                                  | 必选     | SDK 到声网边缘服务器的平均往返时延（Round-Trip Time)，单位 ms。                                                                                                                                                                                                                                                                                         |

## joinOptions

加入频道所需参数。

```typescript
export interface joinOptions {
  appid: string;
  channel: string;
  token: string | null;
  uid?: UID | null;
}
```

| 属性      | 类型                           | 是否必选 | 描述                                                                                                                                                                                                                                                                |
| --------- | ------------------------------ | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `appid`   | `string`                       | 必选     | 声网项目的 App ID。详见 [join](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartcclient.html#join) 的参数说明。                                                                                               |
| `channel` | `string`                       | 必选     | 要加入的频道名称。详见 [join](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartcclient.html#join) 的参数说明。                                                                                                |
| `token`   | `string` &VerticalLine; `null` | 必选     | 用于鉴权的 Token，如果频道启用了 Token 鉴权机制，则需要提供有效的 Token；如果没有启用 Token 鉴权，则传入 `null`。详见 [join](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartcclient.html#join) 的参数说明。 |
| `uid`     | `UID` &VerticalLine;`null`     | 可选     | 用户 ID，如果不提供则由服务器自动生成。详见 [join](https://docportal.shengwang.cn/cn/live-streaming-premium-4.x/API%20Reference/web_ng/interfaces/iagorartcclient.html#join) 的参数说明。                                                                           |

## AgoraRtcReactError

抛出的错误。

```typescript
export class AgoraRtcReactError extends Error {
  public readonly rtcMethod: string;
  public readonly rtcError: AgoraRTCError | string;
  public override readonly name: string = "AgoraRTCReactException";

  public constructor(rtcMethod: string, rtcError: AgoraRTCError | string) {
    if (typeof rtcError === "string") {
      super(rtcError);
    } else {
      super(rtcError.message);
    }
    this.rtcMethod = rtcMethod;
    this.rtcError = rtcError;
  }

  public print(type: printType) {
    if (typeof this.rtcError === "string") {
      console[type](this.rtcError);
    } else {
      console[type](this.rtcError.message);
    }
  }
}
```

| 属性        | 类型                                    | 是否必选 | 描述                       |
| ----------- | --------------------------------------- | -------- | -------------------------- |
| `rtcMethod` | `string`                                | 必选     | 报错的 JavaScript 方法。   |
| `rtcError`  | `AgoraRTCError` &VerticalLine; `string` | 必选     | web 的 class AgoraRTCError |
| `name`      | `AgoraRTCError` &VerticalLine; `string` | 必选     |                            |

## IAgoraRtcReactError

export interface IAgoraRtcReactError extends AgoraRTCError {
  readonly rtcMethod: string;

}
