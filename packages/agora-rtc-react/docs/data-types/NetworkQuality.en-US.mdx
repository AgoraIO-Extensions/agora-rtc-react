## NetworkQuality

The last-mile network quality.

This interface inherits [`NetworkQuality`](https://api-ref.agora.io/en/video-sdk/web/4.x/interfaces/networkquality.html) from the Web SDK and adds the following additional properties:

| Property | Type     | Required | Description                                                                                             |
| -------- | -------- | -------- | ------------------------------------------------------------------------------------------------------- |
| `delay`  | `number` | Yes      | The average Round-Trip Time (RTT) from the SDK to the Agora edge server, measured in milliseconds (ms). |
