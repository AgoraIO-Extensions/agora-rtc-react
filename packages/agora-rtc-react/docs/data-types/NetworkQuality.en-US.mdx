## NetworkQuality

The last-mile network quality.

| Property   | Type                                                                                                                  | Required | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| ---------- | --------------------------------------------------------------------------------------------------------------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `uplink`   | `0` &VerticalLine; `1` &VerticalLine; `2` &VerticalLine; `3` &VerticalLine; `4` &VerticalLine; `5` &VerticalLine; `6` | Yes      | The uplink network quality. It is calculated based on the uplink transmission bitrate, uplink packet loss rate, RTT (round-trip time) and jitter.<ul><li>0: The quality is unknown.</li><li>1: The quality is excellent.</li><li>2: The quality is good, but the bitrate is less than optimal.</li><li>3: Users experience slightly impaired communication.</li><li>4: Users can communicate with each other, but not very smoothly.</li><li>5: The quality is so poor that users can barely communicate.</li><li>6: The network is disconnected and users cannot communicate.</li></ul>   |
| `downlink` | `0` &VerticalLine; `1` &VerticalLine; `2` &VerticalLine; `3` &VerticalLine; `4` &VerticalLine; `5` &VerticalLine; `6` | Yes      | The downlink network quality. It is calculated based on the uplink transmission bitrate, uplink packet loss rate, RTT (round-trip time) and jitter.<ul><li>0: The quality is unknown.</li><li>1: The quality is excellent.</li><li>2: The quality is good, but the bitrate is less than optimal.</li><li>3: Users experience slightly impaired communication.</li><li>4: Users can communicate with each other, but not very smoothly.</li><li>5: The quality is so poor that users can barely communicate.</li><li>6: The network is disconnected and users cannot communicate.</li></ul> |
| `delay`    | `number`                                                                                                              | Yes      | The average Round-Trip Time (RTT) from the SDK to the Agora edge server, measured in milliseconds (ms).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
